Project Case Study

Rally Insurance /
Ubezpieczenia Rajdowe

Insurance website dedicated to rally and motorsport. Together with Rożek Brokers Group we have made research and came up with a brief on which project was based on. We made sure that website is user-friendly and has a great storytelling.

Visit live site

The Challenge

Together with Rozek Brokers Group we designed website dedicated to rally incurance and motorsport. We wanted to deliver perfect solution and came up with dozens of design concepts until we found the one. Webiste is available in two languages - polish and english.

Design from the scrach

Website was designed directed for end users and main audience. We wanted to give a website a character and emphesize the action and dynamic connected to motorsport. We used desktop first approach to create a layout.

Our first task was to give a makeover to RBG logotype.

We used high resolution images for better visual experiences.

We made sure that contact forms are easy accesable and obvious as the last step.

We made sure that user won’t feel lost on website, in such case we desinged a template for all the unexpected errors.

We have set the right grid to show offer in the best possible way.

Mobile Design approach

We made sure that website is well optimised for mobile devices. We hide less important content and stack every section in order to present offer clearly.
